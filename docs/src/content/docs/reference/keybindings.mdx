---
title: Keybindings
description: A reference for all default keybindings in rovr.
---

this page provides a comprehensive list of the default keybindings in `rovr`. you can customize these keybindings in your `config.toml` file.

| action                       | default hotkey                                 | description         |
| ---------------------------- | ---------------------------------------------- | ------------------- |
| toggle_pin                   | <kbd>P</kbd>                                   | Pin folder          |
| toggle_pinned_sidebar        | <kbd>S</kbd>                                   | Toggle sidebar      |
| toggle_preview_sidebar       | <kbd>I</kbd>                                   | Toggle preview      |
| toggle_footer                | <kbd>F</kbd>                                   | Toggle footer       |
| focus_toggle_pinned_sidebar  | <kbd>s</kbd>                                   | Focus sidebar       |
| focus_file_list              | <kbd>escape</kbd>                              | Focus file list     |
| focus_toggle_preview_sidebar | <kbd>i</kbd>                                   | Focus preview       |
| focus_toggle_path_switcher   | <kbd>ctrl+l</kbd>                              | Focus path          |
| focus_toggle_processes       | <kbd>f6</kbd>                                  | Focus processes     |
| focus_toggle_clipboard       | <kbd>c</kbd>                                   | Focus clipboard     |
| focus_toggle_metadata        | <kbd>m</kbd>                                   | Focus metadata      |
| focus_search                 | <kbd>ctrl+f</kbd> <kbd>slash</kbd>             | Search              |
| copy                         | <kbd>y</kbd> <kbd>ctrl+c</kbd>                 | Copy                |
| paste                        | <kbd>p</kbd> <kbd>ctrl+v</kbd>                 | Paste               |
| cut                          | <kbd>x</kbd> <kbd>ctrl+x</kbd>                 | Cut                 |
| delete                       | <kbd>d</kbd> <kbd>delete</kbd>                 | Delete              |
| rename                       | <kbd>r</kbd> <kbd>f2</kbd>                     | Rename              |
| new                          | <kbd>ctrl+n</kbd> <kbd>o</kbd>                 | New                 |
| toggle_all                   | <kbd>%</kbd> <kbd>ctrl+a</kbd>                 | Select all          |
| zip                          | <kbd>E</kbd>                                   | Zip                 |
| unzip                        | <kbd>ctrl+e</kbd>                              | Unzip               |
| copy_path                    | <kbd>Y</kbd>                                   | Copy path           |
| up                           | <kbd>up</kbd> <kbd>k</kbd>                     | Up                  |
| down                         | <kbd>down</kbd> <kbd>j</kbd>                   | Down                |
| up_tree                      | <kbd>left</kbd> <kbd>h</kbd>                   | Go up directory     |
| down_tree                    | <kbd>right</kbd> <kbd>l</kbd> <kbd>enter</kbd> | Enter/Select        |
| page_up                      | <kbd>pageup</kbd> <kbd>ctrl+b</kbd>            | Page Up             |
| page_down                    | <kbd>pagedown</kbd> <kbd>ctrl+f</kbd>          | Page Down           |
| home                         | <kbd>home</kbd> <kbd>g</kbd>                   | First               |
| end                          | <kbd>end</kbd> <kbd>G</kbd>                    | Last                |
| hist_previous                | <kbd>backspace</kbd> <kbd>u</kbd>              | History back        |
| hist_next                    | <kbd>space</kbd>                               | History forward     |
| toggle_visual                | <kbd>v</kbd>                                   | Visual mode         |
| toggle_hidden_files          | <kbd>full_stop</kbd>                           | Toggle hidden files |
| select_up                    | <kbd>shift+up</kbd> <kbd>K</kbd>               | Select up           |
| select_down                  | <kbd>shift+down</kbd> <kbd>J</kbd>             | Select down         |
| select_page_up               | <kbd>shift+pageup</kbd>                        | Select page up      |
| select_page_down             | <kbd>shift+pagedown</kbd>                      | Select page down    |
| select_home                  | <kbd>shift+home</kbd>                          | Select to top       |
| select_end                   | <kbd>shift+end</kbd>                           | Select to end       |
| tab_next                     | <kbd>ctrl+j</kbd>                              | Next tab            |
| tab_previous                 | <kbd>ctrl+k</kbd>                              | Previous tab        |
| tab_new                      | <kbd>n</kbd>                                   | New tab             |
| tab_close                    | <kbd>w</kbd>                                   | Close tab           |
| preview_scroll_left          | <kbd>left</kbd> <kbd>h</kbd>                   | Scroll left         |
| preview_scroll_right         | <kbd>right</kbd> <kbd>l</kbd>                  | Scroll right        |
| preview_select_right         | <kbd>shift+right</kbd> <kbd>L</kbd>            | Select right        |
| preview_select_left          | <kbd>shift+left</kbd> <kbd>H</kbd>             | Select left         |
| show_keybinds                | <kbd>question_mark</kbd>                       | Show keybinds       |

## alternate layers

### `change_sort_order`

Keybinds related to changing the sort order

| action     | default hotkey | description           |
| ---------- | -------------- | --------------------- |
| open_popup | <kbd>,</kbd>   | Open sort order popup |
| name       | <kbd>a</kbd>   | Sort by name          |
| extension  | <kbd>e</kbd>   | Sort by extension     |
| natural    | <kbd>n</kbd>   | Sort naturally        |
| size       | <kbd>s</kbd>   | Sort by size          |
| created    | <kbd>c</kbd>   | Sort by created       |
| modified   | <kbd>m</kbd>   | Sort by modified      |
| descending | <kbd>d</kbd>   | Toggle descending     |

### `delete_files`

Keybinds related to the delete confirmation screen

| action | default hotkey                 | description           |
| ------ | ------------------------------ | --------------------- |
| trash  | <kbd>d</kbd>                   | Delete to recycle bin |
| delete | <kbd>x</kbd>                   | Permanently delete    |
| cancel | <kbd>c</kbd> <kbd>escape</kbd> | Cancel deletion       |

### `filename_conflict`

Keybinds related to handling a conflict with two files of the same name

| action         | default hotkey                 | description        |
| -------------- | ------------------------------ | ------------------ |
| overwrite      | <kbd>o</kbd>                   | Overwrite file     |
| skip           | <kbd>s</kbd>                   | Skip file          |
| rename         | <kbd>r</kbd>                   | Rename file        |
| cancel         | <kbd>c</kbd> <kbd>escape</kbd> | Cancel operation   |
| dont_ask_again | <kbd>a</kbd>                   | Don't ask me again |

### `file_in_use`

Keybinds related to handling a file that is in use by another process

| action         | default hotkey                 | description        |
| -------------- | ------------------------------ | ------------------ |
| retry          | <kbd>r</kbd>                   | Retry operation    |
| skip           | <kbd>s</kbd>                   | Skip file          |
| cancel         | <kbd>c</kbd> <kbd>escape</kbd> | Cancel operation   |
| dont_ask_again | <kbd>a</kbd>                   | Don't ask me again |

### `filter_modal`

Keybinds related to selecting options in a modal screen (like FileSearch or ZDToDirectory)

| action | default hotkey                    | description |
| ------ | --------------------------------- | ----------- |
| exit   | <kbd>escape</kbd>                 | Exit modal  |
| down   | <kbd>down</kbd> <kbd>ctrl+j</kbd> | Select down |
| up     | <kbd>up</kbd> <kbd>ctrl+k</kbd>   | Select up   |

### `yes_or_no`

Keybinds related to yes/no confirmation modals

| action         | default hotkey                 | description        |
| -------------- | ------------------------------ | ------------------ |
| yes            | <kbd>y</kbd>                   | Yes                |
| no             | <kbd>n</kbd> <kbd>escape</kbd> | No                 |
| dont_ask_again | <kbd>a</kbd>                   | Don't ask me again |
